## 项目概述

* 产品名称：laravel5.5-chatroom
* 项目代码：laravel5.5-chatroom
* 演示地址：http://dev10.shareg.cn

laravel5.5-chatroom是一个简洁的聊天室应用，使用laravel5.5编写的聊天室api

## 功能如下
- 用户认证 —— 注册、登录;
- 聊天室发送文字和图片消息
- 接入图灵机器人，可自定义开关

## 运行环境要求

- Nginx 1.8+
- PHP 7.0+
- Mysql 5.7+

## 开发环境部署/安装
